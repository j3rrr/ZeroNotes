Zero Notes

ZeroNotes is designed to help you create notes for your raid, depending on customizable Boss- and Player-Spell settings.

Feature overview:
- DB for Player spells
  - Set ratings for each spells
  - Define stationary / movement and AoE / Single target
  - Setup spelltypes for healing / utility / imunity spells
- Boss Templates
  - Setup spells during an encounter
  - Set priority for each spell
  - Set how many heal / utility / imunity spells you want to assign
  - Advanced Settings to overwrite certain ratings 
- Group Templates
  - Create Group Templates for up to 30 Player
  - Create Templates from your current Group
- Build Notes using a Group Template or your current raid Group
- Import / Export to share your Templates or Playerspell DB
- Send Note directly to Exorsus Raid Tools **(LINK)** or use our additional Weakaura

How does it work?
First you need to setup the Playerspell DB.  
ZeroNotes comes with pre-defined ratings for all basic healing, utility and imunity cooldowns.  
Feel free to edit, remove or add spells however you need them.  

After setting up player spells, you need a Boss Template.  
Setup each spell you want to cover with cooldowns during the encounter.

Time: First time the boss will cast that spell
Prio: Internal priority
Station: Does the spell force you to stand still over can you move during the cast?
AoE: Does the spell effect the entire raid or only one player?
X Times / Interval: How often and when is that spell repeated?
Heal / Utility / Immunity: How many cooldowns of each type do you want to assign?
Edit: Advanced Settings to override player spell ratings (see below)

Additionally, you can setup lines to divide your note (e.g. Phase)

Advanced Settings:
Clicking the Edit Button for a Boss Spell, will open a new Popup
Here you can overwrite playerspell ratings or assign a specific player to that spell.

Example: In "Heal" Row, set "Class" to "Shaman" to not use Aura Mastery but Spirit Link / Healing Tide Totem. Or set the player name to "Superpriest" and if you have a player named "Superpriest" in your group, they will always be assigned to that spell. Don't forget to set "+ Rating" to something high.

IMPORTANT
This will not overwrite the spells priority. Meaning, in our example above, if you edit "Charge" and set a +Rating for Holy Paladins, "Execute" will still get Aura Mastery Assigned.
To avoid that, create a new "Charge" at the time you want Aura Mastery to be assigned and set a higher priority. The same applies if you want to overwrite settings only for the 2nd or 3rd repetition.+
